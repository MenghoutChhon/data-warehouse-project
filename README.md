# 📊 Data Warehouse Project & Analytics Project

Welcome to **Menghout Chhon's Data Warehouse & Analytics Project**! 🚀  
This repository contains a complete data warehousing solution and analytics system to manage, process, and analyze business data efficiently.

---

## 📌 Project Overview

This project showcases the end-to-end development of a Data Warehouse and Analytics system, including:

- 📂 **Data Warehouse Design**
- 🔄 **ETL (Extract, Transform, Load) Pipelines**
- 📈 **Interactive Analytics Dashboards**

---

## 🛠️ Technologies Used

- **Database**: PostgreSQL  
- **ETL Tool**: Pentaho Data Integration (PDI)  
- **Programming**: SQL, Python  
- **Business Intelligence (BI)**: Metabase  
- **Cloud Services**: Google Cloud (BigQuery, Cloud Storage)  
- **Data Sources**: Kaggle datasets, CSV files, and APIs  

---

## 📁 Project Components

### 1️⃣ Data Warehouse Architecture

A clean, optimized schema for storing structured data suitable for reporting and analytics.

> Example tables: `sales`, `customers`, `products`, `transactions`

---

### 2️⃣ ETL Pipeline

- Extracts data from CSVs/APIs
- Transforms data for consistency and usability
- Loads clean data into the PostgreSQL warehouse

---

### 3️⃣ Analytics Dashboards

Created using Metabase to visualize KPIs and business metrics like:
- 📊 Sales trends
- 👥 Customer behavior
- 📦 Product performance  
- 🌍 Geographic trends  

---
